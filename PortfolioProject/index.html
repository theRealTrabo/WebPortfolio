About

I was first introduced to HTML and CSS in high school. While web development fascinated me, I didn't stick with it beyond schoolwork. Fast forward 8 years, and I have rediscovered that fascination while recently learning about free/libre open-source software (FOSS/FLOSS). This sparked a desire to create my own unique projects through the power of coding and specifcally through web development, at least as a starting point.

With that goal in mind, I started over learning HTML and CSS to build a foundation with the help of freecodecamp.org and SmolJames on YouTube. Using those resources, I have been slowly working toward becoming a certified full stack web developer. My goal is to be certified and searching for a job within 4-6 months from the time of writing this in September of 2024.

This very website was created following a tutorial by SmolJames, originally inspired by Brittany Chiang, although I have made some of my own unique design choices in hopes of stretching myself beyond tutorials.
